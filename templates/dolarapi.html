Bienvenido
Esta es una forma auxiliar para la API del valor del dolar.

Instrucciones: por defecto al acceder a dolarapi el servidor devolverá el ultimo valor registrado para el dolar, tanto compra como venta. Podemos cambiar el resultado mediante los siguientes parametros.

  fecha: ej: 1 , 10-01 ,20-10-2030
con este parametro el servidor buscara el valor correspondiente a la fecha y devolverá un error en caso contrario el valor por defecto es la fecha de hoy
  valor: ej: c, v, cv
este parametro decide que valor nos devuelve, c = valor compra, v = valor venta, cv = compra y venta, este ultimo es el valor por defecto y devuelve un diccionario con las claves "compra" y "venta" y sus respectivos valores.
